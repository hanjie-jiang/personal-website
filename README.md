## Personal Website
My [Personal Website](https://hanjie-jiang.github.io/personal-website).

This website built with [Hugo](https://gohugo.io/) and the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme. <br>
It was originally coded with HTML and CSS script from 2022 and revamped using the existing structure in 2026. <br>

**Live site:** [hanjie-jiang.github.io/personal-website](https://hanjie-jiang.github.io/personal-website/)

### Structure

```
content/
├── about.md          # About page
├── experience.md     # Work experience
├── projects.md       # Projects showcase
├── awards.md         # Awards and honors
├── publications.md   # Academic publications
└── posts/            # Blog posts
```

### Local Development

#### Prerequisites

- [Hugo](https://gohugo.io/installation/) (extended version recommended)

#### Setup

```bash
# Clone the repository with submodules
git clone --recurse-submodules https://github.com/hanjie-jiang/personal-website.git

# Or if already cloned, initialize submodules
git submodule update --init --recursive
```
